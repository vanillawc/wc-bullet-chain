## Installation

```sh
npm i @vanillawc/wc-bullet-chain
```

The package can be found in `node_modules/@vanillawc/wc-bullet-chain`.

-----

## Usage

### Load HTML from an external file

```html
<wc-bullet-chain>
  <li>one</li>
  <li>two</li>
  <li>three</li>
  <li>four</li>
</wc-bullet-chain>
```
