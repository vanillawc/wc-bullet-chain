A Vanilla Web Component for creating BBC-style chained unordered lists.

-----

## Installation

```sh
npm i @vanillawc/wc-bullet-chain
```

Then import the `index.js` file at the root of the package.

-----

## Usage

### Load HTML from an external file

```html
<wc-bullet-chain>
  <li>one</li>
  <li>two</li>
  <li>three</li>
  <li>four</li>
</wc-bullet-chain>
```
